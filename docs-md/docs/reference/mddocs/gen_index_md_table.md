Script to generate a Markdown table of links from a Docusaurus `sidebar.json`.

Usage examples:
  python src/mddocs/gen_index_md_table.py     --sidebar docs-md/docs/reference/sidebar.json     --docs-root docs-md/docs     &gt; sidebar_links.md

This script generates a Markdown index file with table of links from a `sidebar.json` generated by `pydoc-markdown docusaurus`.
